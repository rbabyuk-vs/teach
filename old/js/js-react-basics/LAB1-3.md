# Лабораторна робота

## Основи JavaScript: змінні, умови, цикли

### Мета роботи

Навчитися:

* оголошувати та використовувати змінні;
* застосовувати умовні оператори (`if / else`);
* використовувати цикл `for`;
* працювати з числовими значеннями та логікою.

---

## Умови виконання

* Мова: **JavaScript**
* Без використання:

  * функцій
  * масивів
  * `while`
* Виведення результатів через `console.log()`

---

## Завдання

### **Завдання 1. Лічильник чисел**

1. Створи змінну `n` і задай їй значення **10**.
2. За допомогою циклу `for` виведи в консоль **усі числа від 1 до n**.

Очікуваний результат:

```
1
2
3
...
10
```

---

### **Завдання 2. Парні та непарні**

1. За допомогою циклу `for` пройди числа **від 1 до 20**.
2. Для кожного числа:

   * якщо воно **парне** — виведи

     ```
     4 is even
     ```
   * якщо **непарне** —

     ```
     5 is odd
     ```

Підказка: використай оператор `%`.

---

### **Завдання 3. Сума чисел**

1. Створи змінну `sum` зі значенням `0`.
2. За допомогою циклу `for` додай до `sum` **усі числа від 1 до 100**.
3. Виведи фінальне значення `sum` у консоль.

Очікуваний результат:

```
Sum is: 5050
```

---

### **Завдання 4. Перевірка оцінки**

1. Створи змінну `score` (значення від 0 до 100).
2. За допомогою умов:

   * `90–100` → `"Excellent"`
   * `75–89` → `"Good"`
   * `60–74` → `"Satisfactory"`
   * `< 60` → `"Fail"`
3. Виведи результат у консоль.

Приклад:

```
Score: 82
Result: Good
```

---

### **Завдання 5. Таблиця множення**

1. Створи змінну `number` (наприклад, `5`).
2. За допомогою циклу `for` виведи таблицю множення **від 1 до 10**.

Очікуваний результат:

```
5 x 1 = 5
5 x 2 = 10
...
5 x 10 = 50
```
